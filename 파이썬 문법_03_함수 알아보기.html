3. 함수 알아보기¶

  • 함수는 def 예약어를 사용해서 정의한다.
  • 함수는 이름을 지정한다. 이름없는 함수는 lambda 예약어를 사용한다.
  • 함수는 정의하고 호출해서 사용한다.
  • 이름없는 함수는 작성하면 바로 호출해서 사용할 수 있다.
  • 함수 정의할 때 매개변수를 정의하고 함수를 호출할 때 이 매개변수와 매핑되는 인자를 값으로 전달한다.

중첩함수(내부함수) 정의하기¶

  • 함수 내부에 함수를 정의해서 내부함수를 실행한 후에 반환한다
In [33]:
def add_mul(x,y) :
    print('add_mul 실행 ')
    def mul() :
        print('mul 실행')
        return x* y
    return x+y + mul()

재귀함수¶

  • 자기 함수를 내부에서 다시 호출해서 사용하는 함수
  • 재귀함수 작성할 때 마지막 처리 로직을 반드시 정해야 한다. 없을 경우 무한 반복을 처리한다.
  • 재귀함수 호출은 return 문에 작성한다. 인자로 전달하는 값을 항상 줄여서 전달한다.
In [7]:
def fac(num) :
    if num == 0 : # 마지막 호출할 경우 끝으로 처리해서 빠져나가는 로직
        return 1
    return fac(num-1) * num
In [9]:
fac(5)
Out[9]:
120

3-7. 네임스페이스 참조 규칙¶

  • 프로그램언어는 내장 영역이 제일 상위에 있다.
  • 모듈울 작성하면 전역 영역이 생긴다. 모듈에서는 내장 영역을 참조할 수 있다.
  • 함수를 정의하면 지역이 생긴다. 함수 내부에 내부함수를 정의하면 내부함수 내부도 자신의 지역영역이다.

모듈에 정의된 변수는 전역(global ) 영역이고 함수 내의 변수는 지역(local) 영역¶

  • 함수를 실행하면 내부변수부터 참조
In [13]:
var = 100

def func() :
    var = 200
    return var
In [14]:
func()
Out[14]:
200

함수 내부에서 전역변수를 사용할 때는 반드시 global 예약어를 지정해야함¶

In [15]:
def func_() :
    global var
    return var
In [16]:

Out[16]:
100

함수 내부에 함수를 정의하면 내부함수는 외부함수의 모든 변수를 공유해서 사용함¶

In [23]:
def outer() :
    var = 300
    def inner() :
        return var
    return inner()

내부함수에서 외부함수의 변수를 지정해서 사용할 경우¶

def outer_() :
    var_ = 500
    def inner_() :
        nonlocal var_
        return var_
    return inner_()
