모듈 내에 정의된 함수 등을 확인¶

  • dir 함수로 모듈을 확인하면 내부에 가진 모든 것을 문자열로 변환한다.
  • 그중에 스페셜 속성이나 메소드는 제외

count = 0 
for i in dir(op) :
    if not i.startswith("_")  :

        count += 1
        if count % 5 ==0 :

내부에 덧셈연산을 하는 add 스페셜 메소드 여부 확인¶

  • 클래스 내에서 스페셜 메소드가 없는 경우 예외가 발생한다.
  • 예외는 try except 구문으로 처리한다.
In [20]:
def add(obj1, obj2) :
    try :
        if (obj1).__class__.__dict__['__add__'] :
            result = op.add(obj1,obj2)
    except KeyError as e :
        result = e
    return result

정수, 문자열, 튜플, 리스트에는 덧셈 연산자가 있다¶

파이썬 패키지로 인식하도록 init.py를 만든다.¶

  • all 이라는 리스트 변수가 있는데, 이 변수는 "from 패키지명 import *" 문을 사용할 때, 그 패키지 내에서 import 가능한 모듈들의 리스트를 담고 있다. 즉, all 에 없는 모듈은 import 되지 않고 에러가 발생한다.
