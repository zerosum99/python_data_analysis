4. 클래스 알아보기¶

  • 클래스는 사용하고자 하는 객체들을 만들기 위한 틀로 정의한다.
  • 클래스를 정의한 후에 객체를 만들어서 실제 객체를 사용한다.

4-1 내장 클래스 알아보기¶

  • 파이썬 내부에서 제공하는 기본 클래스들

정수 클래스 확인하기¶

  • type 클래스내에 클래스를 넣고 확인
  • 객체를 가지고 내부의 class 속성으로 확인하기
In [1]:

Out[1]:
type
In [2]:

In [3]:

  • 이 클래스는 사용자 클래스를 만들 때 최상위 클래스 역할을 함
  • 이 클래스는 객체만 만들 수 있지만 특별한 속성이 없음

4-5 사용자 클래스의 메소드 넣기¶

  • 객체가 사용할 수 있는 메소드를 정의할 때 첫번째 매개변수는 항상 self를 넣는다.
  • 실제 객체가 self 자리에 들어와서 메소드를 처리한다.

클래스를 정의할 때 두 개의 속성과 두 개의 메소드를 정의한다.¶

  • 메소드는 단순하게 조회만 할 수 있도록 내부에 매개변수는 self 만 넣는다
In [31]:
class Person :
    def __init__(self, name, age) :
        self.name = name
        self.age = age
        
    def getName(self) :
        print(" getName 실행 ")
        return self.name
    
    def getAge(self) :
        print(" getAge 실행")
        return self.age

객체를 생성한다¶

In [32]:
p_1 = Person("이름", 24)

메소드 호출 방법¶

  • 객체 다음에 점연산자와 메소드 이름을 작성한다.
  • 메소드를 실행 연산자 안에 인자를 전달한다.
  • self는 객체가 자동으로 세팅되므로 아무런 인자도 전달할 필요가 없다
